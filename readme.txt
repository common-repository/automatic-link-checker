Removes broken links and store them

== Description ==

This plugin scans your blog posts for any links and removes any links which are no longer indexed in Google, or where the site is down.

== Installation ==

1. Upload `Automatic_link_checker` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Can I restore removed link? =

Yous, you can in wordpress backend in plugins' settings
